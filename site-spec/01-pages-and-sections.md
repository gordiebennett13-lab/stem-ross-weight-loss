# 01 — Pages & Sections

The structural skeleton of a lead-gen site: what pages must exist, and what
sections live inside each. The whole site has **one job — turn a visitor into a
booked lead.** Every section should push toward that.

## Required pages

| Page | Purpose | Ship in v1? |
|---|---|---|
| **Home** | The pitch + primary conversion path | Required |
| **Services / Offer** | What they sell, in detail | Required (can be a section on Home for a one-pager) |
| **About** | Who they are, why trust them | Required |
| **Contact / Book** | The conversion page — form + booking + NAP + map | Required |
| **Thank-you / confirmation** | Post-submit page (fires conversion event, sets expectations) | Required |
| **404** | Branded not-found with a way back | Required |
| **Privacy Policy** | Legal — required if you run any tracking/forms | Required |
| **Terms of Service** | Legal | Required |
| **Accessibility statement** | ADA cover | Recommended |

### Optional pages
- **Results / Testimonials** — dedicated proof page (or a Home section).
- **Blog / Resources** — only if the client will actually maintain it.
- **Individual service pages** — one per service when services are distinct
  enough to need their own SEO landing page.

## Home page — section anatomy (in order)

1. **Sticky nav** — logo left, 3–5 links, primary CTA button right. Shrinks/
   solidifies on scroll. Mobile collapses to a drawer.
2. **Hero** — headline (the promise), subhead (how/for whom), primary CTA,
   supporting image or video, optional trust badge (rating, years, # served).
3. **Social-proof bar** — star rating, review count, client/partner logos, or a
   "trusted by N+" stat. Immediately after the hero to build credibility early.
4. **Problem / agitation** — name the pain the visitor feels. Short.
5. **Services / offer cards** — what they provide, 3-column card grid, each with
   icon, title, one line, optional link to detail.
6. **How it works** — 3 simple steps (e.g. Book → Plan → Results). Removes
   friction by showing the path is easy.
7. **Results / before-after / testimonials** — proof it works. Quotes with name +
   photo, star ratings, before/after where relevant (see niche overlay).
8. **About / credentials** — who's behind it, qualifications, real photos.
9. **Offer / "free consult"** — the explicit ask: free consultation, intro offer,
   pricing if it converts. One clear thing to do next.
10. **FAQ** — handle the top 5–8 objections (cost, time, insurance, "is it for me").
11. **Final CTA + lead form** — repeat the primary action with the form inline so
    they don't have to navigate away.
12. **Footer** — NAP (name, address, phone), hours, social links, nav, legal
    links, copyright.

### Must-have vs nice-to-have for a v1 ship
- **Must:** nav, hero, social proof, services, how-it-works, one proof element,
  offer/CTA, lead form, footer.
- **Nice-to-have:** problem section, FAQ, dedicated about section (can fold into
  hero/offer on a tight one-pager).

## Cross-page rules
- A primary CTA is visible on every page (nav + at least one in-page).
- The lead form / booking path is reachable from every page in one click.
- NAP is consistent everywhere (must match Google Business Profile exactly — see
  `05` local SEO).
